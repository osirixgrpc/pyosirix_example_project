# pyOsiriX Example Project
An example project to demonstrate a method for creating a full project for use in pyOsiriX.

It uses the following project structure:

```text
pyosirix_example_project/
├── package_1/
│   ├── __init__.py
│   ├── p1_module_1.py
│   └── p1_module_2.py
├── package_2/
│   ├── __init__.py
│   ├── p2_module_1.py
│   └── p2_module_2.py
├── .gitignore
├── LICENSE
└── README.md
```
